# DESIGN.md — Global Premium Visual System

## 1. Design Philosophy

- Cinematic, not decorative
- Emotional, not flashy
- Minimal surface, maximum depth
- Every section must have a purpose
- Beauty comes from contrast, not clutter

Golden rule:
→ If everything is loud, nothing stands out.

---

## 2. Visual Direction

- Dark + light contrast driven
- Hero = immersive, cinematic
- Sections = alternating rhythm (dark / light)
- Focus-driven layouts (one focal point per section)

Avoid:
- flat boring sections
- repetitive blocks
- generic UI patterns

---

## 3. Color System

Primary:
- Deep emerald (#0B3D2E)
- Near black (#071A14)

Accent:
- Soft gold (#D4AF37)
- Subtle glow green (#2ECC71)

Support:
- Pure white (#FFFFFF)
- Neutral gray (#F3F4F6)

Rules:
- Use gradients VERY selectively
- Prefer depth (shadows, overlays) over color noise
- Keep contrast strong

---

## 4. Typography (High Impact)

Headlines:
- Bold, large, dominant
- Tight line-height
- Emotional tone

Body:
- Clean, readable
- Comfortable spacing

Rules:
- One hero message must dominate screen
- Bengali readability must be perfect
- Avoid weak, small headings

---

## 5. Hero Section (Signature Experience)

Must include:

- Massive headline (dominant)
- Clear message (no ambiguity)
- Strong CTA (visible instantly)
- Visual focal point (map/globe)

Visual direction:

- Realistic Bangladesh map/globe
- Subtle glow around Bangladesh
- Soft atmospheric lighting
- Depth via overlay gradients (not loud colors)

Rules:

- Hero must stop scrolling
- Message must be understood in 2 seconds
- No abstract meaningless graphics

---

## 6. Depth & Layers

Use:

- soft shadows
- subtle glow
- layered backgrounds
- gradient overlays (low opacity)

Avoid:

- flat UI everywhere
- heavy blur/glass effects
- noisy textures

---

## 7. Cards (Premium Feel)

- Elevated surface
- soft shadow
- clean border
- generous padding

Hover:

- slight lift
- shadow increase
- subtle highlight

Rules:

- cards must feel tactile
- no generic boxes
- no clutter inside cards

---

## 8. Layout Rhythm

Pattern:

- hero (dark, immersive)
- section (light, calm)
- section (dark, narrative)
- section (light, structured)

Rules:

- never repeat same layout twice
- create visual journey
- guide user attention

---

## 9. CTA (Call to Action)

- strong contrast
- clear hierarchy
- minimal text

Primary:
- solid button

Secondary:
- outline button

Rules:

- CTA must stand out instantly
- never hide CTA in weak contrast

---

## 10. Motion & Interaction

- subtle, smooth, fast
- no heavy animation
- no delay-heavy transitions

Examples:

- fade-in
- slight slide
- hover lift

Rules:

- motion should support clarity
- not distract

---

## 11. Footer (Premium Closure)

- clean layout
- strong contrast
- minimal but structured

Include:

- identity
- key links
- short message

Rules:

- no clutter
- no overloaded columns

---

## 12. Global Identity

Tone:

- serious
- confident
- authoritative
- national + global

Avoid:

- startup gimmicks
- playful UI
- excessive color usage

---

## 13. Performance Rules

- fast loading
- lightweight UI
- no heavy libraries for visuals

---

## 14. Emotional Layer

Each section should evoke:

- pride
- trust
- clarity
- purpose

Not:

- confusion
- noise
- distraction

---

## FINAL RULE

If something:

- looks flashy but unclear ❌
- reduces readability ❌
- slows the site ❌
- distracts from message ❌

→ Remove it.
